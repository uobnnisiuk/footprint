# DEC-0001: T3（災害モード）発動ルール

## 日付
2026-02-01

## 決定内容（SR）

T3発動ルールを「手動フラグを最上位の確定トリガ」＋「公的アラートで自動ON」＋「エリア自動判別は補助シグナル」のハイブリッド三段構成とする。

- T3状態を3値に分離: OFF / RECOMMENDED / ON
- 発動優先順位を固定（手動 > 公的アラート > ジオフェンス > ヒューリスティック）
- TTLによる自動延長/自動失効を導入
- 不変条件 IF-T3-001〜004 を定義

## 理由

- 災害時は通信断・混乱で「自動だけ」は欠落しやすく、逆に「手動だけ」は押せないケースがある
- 確度の高い外部シグナル（公的アラート）が取れる時だけ自動ONにし、曖昧な判定は「推奨」に留めることで、救助価値と誤作動耐性のバランスを取る
- T3の挙動自体を「侵襲度が上がりすぎない」ように分離（確定/推奨）することで、発動条件を厳しくしても守り切れないプライバシー問題を緩和
